# Imágenes de Proyectos

Coloca las imágenes de tus proyectos en esta carpeta.

## Recomendaciones:

- **Formato:** JPG o PNG
- **Tamaño recomendado:** 800x600 px o 16:9 aspect ratio
- **Peso:** Optimiza las imágenes (máximo 500KB cada una)
- **Nombres:** project1.jpg, project2.jpg, project3.jpg, etc.

## Ejemplo de estructura:

```
public/
  projects/
    project1.jpg
    project2.jpg
    project3.jpg
    project4.jpg
    project5.jpg
    project6.jpg
```

Las imágenes aparecerán automáticamente en la sección de proyectos.

Si no tienes imágenes, se mostrará un placeholder con un icono.
